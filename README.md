# Exam Duty Management System

## Overview

The Exam Duty Management System is a Python command-line application that helps manage teachers' examination duty schedules. It reads duty information from an Excel file, allows users to search schedules, find possible duty swaps, and export swap recommendations to Excel.

## Features

- Read exam duty data from an Excel file
- Display all teachers
- View an individual teacher's duty schedule
- Search teachers by duty type
- Find possible reserve-duty swaps
- Export swap details to an Excel file
- Interactive menu-driven interface

## Technologies Used

- Python 3
- Pandas
- Microsoft Excel (.xlsx)

## How to Run

1. Install Python 3.
2. Install Pandas:

```bash
pip install pandas
```

3. Run the program:

```bash
python3 exam_duty_system.py
```
